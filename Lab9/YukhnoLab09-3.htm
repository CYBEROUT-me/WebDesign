Про використання класів

Клас "menu" має такі характеристики: жирний шрифт Arial Black розміром 16pt, білий текст на синьому фоні, великі літери та відступи для зручності натискання. При наведенні курсору елементи меню змінюють свій вигляд, створюючи інтерактивний досвід для користувача.

Примітка: Клас menu використовується для основного навігаційного меню

Клас "small" призначений для відображення метаінформації, такої як дата публікації, ім'я автора, адреса сторінки тощо. Він має менший розмір шрифту (10pt), курсивне накреслення та сірий колір для того, щоб не відволікати увагу від основного контенту.
