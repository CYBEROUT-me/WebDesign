Корисні веб-ресурси

Google
Google — один із популярних в Україні пошукових серверів. Пропонує швидкий та якісний пошук інформації, зображень, відео та інших матеріалів в Інтернеті.

Gmail
Gmail — безкоштовна поштова служба від Google. Надає 15 ГБ безкоштовного сховища, захист від спаму та зручний інтерфейс для роботи з електронною поштою.

YouTube
YouTube — найбільший відеохостинг у світі. Тут можна знайти навчальні матеріали, музику, фільми та мільйони інших відео від користувачів з усього світу.

GitHub
GitHub — платформа для розробників та хостингу коду. Використовує систему контролю версій Git та є найбільшим сховищем відкритого програмного забезпечення.

Stack Overflow
Stack Overflow — найбільша спільнота програмістів для обміну знаннями. Тут можна знайти відповіді на питання з програмування та отримати допомогу від експертів.

Wikipedia
Wikipedia — вільна багатомовна онлайн-енциклопедія. Містить мільйони статей на різні теми, створених спільнотою волонтерів з усього світу.

Udemy
Udemy — популярна платформа онлайн-навчання. Пропонує тисячі курсів з різних тем: програмування, дизайн, бізнес, мови та багато іншого.

Behance
Behance — провідна платформа для дизайнерів та креативних професіоналів. Тут можна публікувати своє портфоліо, знаходити натхнення та встановлювати професійні контакти.
