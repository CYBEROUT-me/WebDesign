========================================
ВИКОРИСТАНІ ТЕХНОЛОГІЇ:
========================================

✓ HTML тег <map> для створення навігаційних карт
✓ HTML тег <area> для визначення інтерактивних областей
✓ Атрибути shape: rect, circle, poly, default
✓ Атрибут coords для координат областей
✓ Атрибут href для посилань
✓ Атрибут title для спливаючих підказок
✓ Атрибут usemap для зв'язку зображення з картою
✓ SVG для створення графічних зображень
✓ Base64 кодування для вбудовування SVG у img src

========================================
ТИПИ ВИКОРИСТАНИХ ОБЛАСТЕЙ:
========================================

1. RECT (прямокутник)
   - coords="x1,y1,x2,y2"
   - Координати лівого верхнього та правого нижнього кутів

2. CIRCLE (коло)
   - coords="x,y,r"
   - Координати центру та радіус

3. POLY (багатокутник)
   - coords="x1,y1,x2,y2,x3,y3,..."
   - Список координат вершин
